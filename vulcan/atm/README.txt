This folder contains atm input files, with the following defeaut files:

atm_HD189.txt   (HD 189733b dayside-average T-P profile)
atm_HD209.txt   (HD 209458b: dayside-average T-P profile)
atm_HD189_Kzz.txt   (HD 189733b: dayside-average T-P-Kzz profile)

The T-P (Kzz) profiles are adopted (interpolated for different vertical resolutions) from Moses et al. (2011)
Details can be found in http://iopscience.iop.org/article/10.1088/0004-637X/737/1/15/meta
